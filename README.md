# Конвертер валют

## Установка проекта
```
npm install
```

### Сборка проекта для разработки  
####(не минифицированная, со вспомогательными средствами)
```
npm run serve
```

### Сборка для развёртывания проекта
####(минифицированная, без вспомогательных средств)
```
npm run build
```

### Проверка стиля кода и пунктуации
```
npm run lint
```
